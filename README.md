# AI-Powered Full-Stack Interview Prep

This repository contains an automated system designed to generate high-quality, technically accurate answers to common Full-Stack Developer interview questions. The system leverages the Gemini API to provide detailed responses formatted in LaTeX, ensuring mathematical and technical notation is rendered perfectly.

## üöÄ How It Works

The core of the project is an automated pipeline that processes a curated list of interview questions:

1. Storage: All questions are stored in a coding-questions.json file.
2. Automation: A Flask-based background job runs every 30 minutes (configurable).
3. Processing: The job scans the JSON file for an object where the answer field is empty.
4. AI Integration: It sends the question to the Gemini AI model with a specific prompt to generate the answer in LaTeX format.
5. Persistence: The generated answer is saved back into the coding-questions.json file, updating the existing object.

## üìÑ Generating the Final LaTeX File

Once your questions have been answered by the AI, you can generate a single, formatted `.txt` document containing all the content.

1. Inside `.file_utils.py`, run the generation function:
   `json_to_latex()`, by providing the path for the output file

This script iterates through coding-questions.json, wraps the content in a LaTeX document class (e.g., article), and exports everything to a file.

## üìù Configuration

You can adjust the frequency of the generation job directly in the app.py file. By default, it is set to run every 30 minutes to respect API rate limits and allow for gradual database building.
